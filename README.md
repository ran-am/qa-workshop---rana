# QA Workshop

React + TypeScript application bootstrapped with [Vite](https://vite.dev).

## Prerequisites

- **Node.js** >= 23
- **Yarn** (v1 classic)

## Getting Started

```bash
yarn install
yarn dev
```

## Scripts

| Command        | Description                          |
| -------------- | ------------------------------------ |
| `yarn dev`     | Start the dev server                 |
| `yarn build`   | Type-check and build for production  |
| `yarn preview` | Preview the production build locally |
| `yarn lint`    | Run ESLint                           |
